Connect the lead engine

Live lead research runs on an Anthropic API key set on your deployment (env var ANTHROPIC_API_KEY). This is a developer key from the Anthropic Console — separate from any Claude subscription, billed pay-as-you-go.

  1. Open the Anthropic Console → API Keys and create a key (sk-ant-…).
  2. Add a payment method under Billing and top up a small amount of credit.
  3. Set ANTHROPIC_API_KEY as an environment variable on your deployment, then redeploy.
  4. Optional: add Apollo as APOLLO_API_KEY for richer contact details.
Get your API key ↗ Full setup guide: idealone-hunter/README.md
